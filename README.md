# ConvocArte.github.io
Plataforma web para conectar artistas con organizadores de eventos (demo interactiva)

ConvocArte es una plataforma web desarrollada en HTML, CSS y JavaScript cuyo objetivo es conectar artistas con empresas u organizadores que buscan talento para eventos culturales, musicales, escénicos y creativos.

Permite que un artista encuentre convocatorias activas, se postule fácilmente y gestione sus solicitudes, mientras que el organizador puede crear eventos, revisar postulaciones y administrar artistas seleccionados.

El sistema funciona completamente en el navegador utilizando LocalStorage como base de datos, sin necesidad de backend esto es provisional debido a que es una demo nada mas :3.
